Welcome to the Geomajas show-case page. Know that this is work in progress.

The goal of this showcase is two-fold:

  • For developers, this show-case acts as a test-case where all the functionalities can be tested. Also for new developers, the source code and XML configuration files are directly available.
  • For users, this show-case acts as a ...well show-case. It demonstrates the many functionalities the Geomajas supports.

For showcases regarding non-GIS related widget functionality, go to the SmartGWT showcase page.

If you find any bugs in this showcase, please report them in the Geomajas issue tracker.
