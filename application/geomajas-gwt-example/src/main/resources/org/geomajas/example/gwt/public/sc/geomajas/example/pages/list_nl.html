Welkom op de Geomajas show-case pagina. Deze pagina is nog volop in ontwikkeling.

Deze applicatie heeft 2 doelen:

  • Voor ontwikkelaars geldt deze applicatie als een test voor de verschillende functionaliteiten. Ook kunnen nieuwe ontwikkelaars de broncode en de XML configuratie bestanden bekijken.
  • Voor eindgebruikers geldt deze applicatie effectief als een show-case, waarbij men een overzicht krijgt van de verschillende functionaliteiten die door Geomajas worden ondersteund.

Voor meer showcases omtrent niet-GID gerelateerde widget functionaliteiten, kan je altijd naar de SmartGWT showcase pagina gaan.

Indien je toevallig bugs zou vinden in deze applicatie kan je deze altijd rapporteren op de Geomajas issue tracker.
