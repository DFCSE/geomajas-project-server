Welcome to the Geomajas dojo example application !

Attributes & Features

The first sample will get you introduced into the basics of Geomajas navigation, vector features, and their attributes.

Start demo!

The maps used in this example are the OpenStreetMap raster layer as background and a vector layer displaying the 10 Belgian provinces and the Brussels capital region.

Editing

This sample introduces you to the editing of the geometries. This includes creating donut polygons, snapping, splitting of polygons, etc.

Start demo!

The background map used in this demo is the OpenStreetMap raster layer. The vector layers are roads and structures in a place called Duisburg, Belgium.

Search & filtering

This sample shows how searching and filtering can be done on the attributes. Installing a filter on a layer, will restrict the available data, while the search mechanism uses the attributes to create search criteria.

Start demo!

This demo will display the countries and major cities of Africa on top of the NASA Blue Marble raster layer.

Technology demo

This sample makes heavy use of dojo widgets to demonstrate how Geomajas can become a full desktop application inside a browser. You will have the ablity to change the vectorlayer styles in real-time.

Start demo!

The background map used in this demo is the NASA Modis raster layer. The vector layers display the main cities, rivers, lakes and climate belts of Africa.
