Geomajas demo hints

This demo shows how the use of separate windows, and a taskbar (see bottom) can give you the impression of working in a full operating system.

Just open one of the panels below, to start working.

This demo provides the possibility to change the styles associated with vector layers.

Just op the style panel, then select a layer and a style, and change it.

You will notice that the style on the map as well as the legend is immediately updated.

The movie you see here, requires the adobe flash plugin to play, and will open in a separate window.
