How do I create a new feature?

Prerequisites

  1. Enable editing mode () in the toolbar.
  2. Select the layer you want to create in. This can be done in the layer control (left). Select for example the "Structures" layer.
    Hint: you can only create new features in vector-layers.

Creating a new feature

  1. Start: Simply click the RIGHT mouse button on the map, and a menu will appear. At the top of this menu, you will see "Create new feature". Select this option to start.
  2. Create: After the "Create new feature" option has been selected, just start clicking to place new points on the map.
  3. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save feature".

How do I edit an existing feature?

Prerequisites

  1. Enable editing mode () in the toolbar.
  2. Select the feature you want to edit on the map (left mouse click).
    Hint: by selecting a feature, it will become yellow.

Editing the selected feature

  1. Start: Simply click the RIGHT mouse button on the map, and a menu will appear. Select "Start editing" to start.

    Hint: as long as the orange edges and points are drawn, the feature is in editing mode.
  2. Drag a point: You can now drag the endpoints of the geometry to other locations.
  3. Insert a point: Click the RIGHT mouse button on one of the orange edges. Then select "Insert point".
  4. Remove a point: Click the RIGHT mouse button on one of the endpoints. Then select "Remove point".
  5. Undo last step: Click the RIGHT mouse button, then select "Undo last step". This will make your last change undone.
  6. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save feature".

How do I create a donut polygon?

Prerequisites

  1. Enable editing mode () in the toolbar.
  2. Select the feature you want to edit on the map (left mouse click).
    Hint: by selecting a feature, it will become yellow.

Editing the selected feature, to create a hole in it

  1. Start: Simply click the RIGHT mouse button on the map, and a menu will appear. Select "Start editing" to start.

    Hint: as long as the orange edges and points are drawn, the feature is in editing mode.
  2. Create hole: Click the RIGHT mouse button, and a menu will appear. Select "Create hole".
  3. Place points: Click new points inside the feature's border.
  4. Stop inserting: Click the RIGHT mouse button, and select "Stop inserting".
    Hint: you are now in normal editing mode. This means you can drag points around, etc.
  5. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save feature".

How do I split a polygon?

Prerequisites

  1. Enable split-polygon mode () in the toolbar.
  2. Select the polygon feature you want to split on the map (left mouse click).
    Hint: by selecting a feature, it will become yellow.

Split the polygon

  1. Start: Simply click the RIGHT mouse button on the map, and a menu will appear. Select the "Start splitting" option to start.
  2. Create split-line: Start clicking to place new points on the map. The line you create here will split the selected polygon.
  3. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save splitting".

General hints

  • The toolbar allows for only 1 tool to be active at any given moment.
    A tool is active when it's color in the toolbar becomes dark grey.
    In other words, when you are editing(), and you select the pan-tool(), editing will be disabled.
  • Sometimes you are editing a feature, that lies partially off the screen.
    Click the RIGHT mouse button while editing, and select "Zoom to fit" to center around the feature.
